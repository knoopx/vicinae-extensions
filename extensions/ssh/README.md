# SSH Extension

Quickly connect to SSH hosts from your `~/.ssh/known_hosts` file.

## Features

- Lists all SSH hosts you've previously connected to
- Connect to hosts with a single click
- Shows hosts that have been verified and stored

## Usage

1. Connect to SSH hosts using the standard `ssh` command
2. Launch the extension and select a previously connected host

## Requirements

- SSH client installed
- Terminal configured to handle SSH connections
